\hypertarget{md_README_autotoc_md5}{}\doxysection{A quoi servent les séquences d’échappement ?}\label{md_README_autotoc_md5}
Une séquence d\textquotesingle{}échappement est une combinaison de caractères qui a une signification autre que les caractères littéraux qu\textquotesingle{}elle contient Elle est marquée par un ou plusieurs caractères précédents ~\newline
 (source \+: \href{https://en.wikipedia.org/wiki/Escape_sequence}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Escape\+\_\+sequence}})

Exemple \+:
\begin{DoxyItemize}
\item \textbackslash{}\textquotesingle{} apostrophe
\item \textbackslash{}" guillemet. Valeur hexadécimale \+: 0x22 ;
\item \textbackslash{}? point d\textquotesingle{}interrogation. Valeur hexadécimale \+: 0x3f ;
\item \textbackslash{} Barre oblique inversée. Valeur hexadécimale \+: 0x5c ;
\item \textbackslash{}0 caractère nul. Valeur hexadécimale \+: 0x00 ;
\item \textbackslash{}a Caractère d\textquotesingle{}appel audible. Valeur hexadécimale \+: 0x07 ;
\item \textbackslash{}b retour arrière (backspace). Valeur hexadécimale \+: 0x08 ;
\item \textbackslash{}f nouvelle page. Valeur hexadécimale \+: 0x0c ;
\item \textbackslash{}n nouvelle ligne. Valeur hexadécimale \+: 0x0a ;
\item \textbackslash{}r retour chariot. Valeur hexadécimale \+: 0x0d ; 
\end{DoxyItemize}
