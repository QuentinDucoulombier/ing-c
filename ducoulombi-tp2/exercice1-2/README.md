## Comment compiler ?

Pour compiler il suffit de se mettre dans le repertoire courant du fichier helloWorld.c

> gcc -Wall exo1.c -o exe  
> ./exe


## Comment generer Doxygen ?

Pour generer doxygen il existe deux solutions : 
La premiere manière

> doxygen -g
> doxygen Doxyfile

ou sinon 

> doxywizard
